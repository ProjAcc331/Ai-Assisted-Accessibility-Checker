# Ai-Assisted-Accessibility-Checker
This is an AI- AI-assisted, educationally focused web accessibility checker. This project relies on 2 Application Programming Interfaces (WAVE and Google Gemini)
The WAVE API has limited monthly tokens; once those tokens run out, they won't replenish until the month resets.
In order to use, simply copy & paste code into any code space like Visual Studio Code or even https://www.onlinegdb.com/#
